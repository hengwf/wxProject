# 医考通 - 专业医学考试题库小程序

## 一、产品定位

**产品名称**：医考通

**一句话描述**：专注执业医师、药师、护士等医疗从业者考试的垂直题库工具

**核心卖点**：
- 海量真题题库
- 智能错题本
- 模拟考试
- 随时随地刷题

## 三、功能规划

### 3.1 核心功能（必须做）

#### 3.1.1 题库模块
- **分类管理**：按科目、章节、难度分类
- **题目类型**：单选题、多选题、判断题、案例分析题
- **题目详情**：题干、选项、答案、解析、知识点标签
- **题目收藏**：用户可收藏题目

#### 3.1.2 刷题模块
- **顺序练习**：按章节顺序刷题
- **随机练习**：随机抽取题目
- **背题模式**：先看答案后做题
- **答题反馈**：答对/答错即时显示解析

#### 3.1.3 模拟考试
- **全真模拟**：按真实考试规格（时间、题量、分值）
- **智能组卷**：根据薄弱点自动生成试卷
- **成绩记录**：保存每次模拟成绩和排名

#### 3.1.4 错题本
- **自动收录**：答错的题目自动进入错题本
- **专项训练**：针对错题反复练习
- **错题统计**：分析错误率最高的知识点

#### 3.1.5 学习进度
- **进度可视化**：已完成/未完成 清晰展示
- **能力图谱**：各知识点掌握程度雷达图
- **目标设定**：每日刷题目标

### 3.2 高级功能（付费权益）

| 功能 | 免费版 | 付费会员 |
|-----|-------|---------|
| 题库数量 | 500题 | 5000+题 |

## 八、技术方案

### 8.1 复用现有框架

基于现有 `wxProject-main` 小程序框架进行改造：

```
现有 → 医考通

pages/index      → 首页（题库分类入口）
pages/levels    → 章节列表
pages/quiz      → 刷题页
pages/result    → 结果页
pages/mine      → 个人中心/会员

utils/questionManager → 适配多分类题库
utils/adManager      → 保留激励视频广告

data/questions.js    → 医学题库
```

### 8.2 后台管理系统

扩展 `admin/server.js`：

```javascript
// 新增API
GET  /api/categories      // 获取所有科目
GET  /api/chapters/:category  // 获取章节
POST /api/exam/submit     // 提交试卷
GET  /api/progress/:userId    // 获取学习进度
POST /api/member/verify   // 会员验证
```

### 8.3 数据库设计（可选）

如果数据量大，可引入数据库：

| 表名 | 说明 |
|-----|------|
| questions | 题目表 |
| categories | 科目表 |
| chapters | 章节表 |
| users | 用户表 |
| progress | 学习进度 |
| members | 会员表 |
| wrong_questions | 错题表 |
