# 益智闯关微信小程序

## 项目说明
这是一个完整的微信益智答题闯关小程序项目，包含题库系统、闯关机制、用户数据记录等功能。

## 问题修复记录

### 已修复的问题：
1. ✅ 题库文件导出格式问题 - 已修复为 `module.exports` 直接导出
2. ✅ tabBar 图标损坏 - 暂时移除了损坏的图标配置

## 运行方式

### 微信小程序端：
1. 使用微信开发者工具导入项目根目录 `/workspace/`
2. 配置小程序 AppID
3. 点击编译运行

### 后台管理系统：
```bash
cd admin
npm install
npm start
# 访问 http://localhost:3000
```

## 项目结构
```
/workspace/
├── app.js                 # 小程序入口
├── app.json               # 小程序配置
├── app.wxss               # 全局样式
├── data/
│   └── questions.js       # 题库数据（200+道题目）
├── pages/                 # 页面文件
└── admin/                 # 后台管理系统
```

## 主要功能
- 四大分类题库（生活常识、影视娱乐、历史人文、综合知识）
- 闯关解锁机制
- 答题进度记录
- 答案解析和提示
